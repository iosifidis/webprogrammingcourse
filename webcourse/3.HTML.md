# Εισαγωγή στην HTML

Η **HTML** (Hypertext Markup Language) είναι μια περιγραφική γλώσσα που χρησιμοποιείται για τη δημιουργία ιστοσελίδων. Με την HTML, μπορούμε να καθορίσουμε τη δομή και την εμφάνιση των στοιχείων μιας ιστοσελίδας, όπως κείμενο, εικόνες, ήχος και πολυμέσα.

## Βασικές Έννοιες της HTML
- **Στοιχεία (Elements):** Τα στοιχεία είναι η βασική μονάδα της HTML και περιγράφονται από ετικέτες (tags).
- **Ετικέτες (Tags):** Οι ετικέτες περιβάλλουν το περιεχόμενο και ορίζονται με `<tag>` στην αρχή και `</tag>` στο τέλος.
- **Attributes (Χαρακτηριστικά):** Παρέχουν πρόσθετες πληροφορίες για τα στοιχεία, όπως `id`, `class`, και `style`.

## Βασική Δομή μιας HTML Σελίδας

```html
<!doctype html>
<html>
<head>
    <meta charset="utf-8">
    <title>Τίτλος Ιστοσελίδας</title>
</head>
<body>
    <h1>Καλώς ήρθατε</h1>
    <p>Αυτή είναι μια απλή ιστοσελίδα.</p>
</body>
</html>
```

### Επεξήγηση
1. `<!doctype html>`: Δηλώνει ότι το έγγραφο χρησιμοποιεί HTML5.
2. `<html>`: Η κύρια ετικέτα που περιλαμβάνει όλο το περιεχόμενο της σελίδας.
3. `<head>`: Περιέχει μεταπληροφορίες (metadata) για τη σελίδα.
4. `<body>`: Περιέχει το περιεχόμενο που προβάλλεται στον χρήστη.

## Κύριες Ετικέτες της HTML

### Δομή και Επικεφαλίδες
- **Επικεφαλίδες:**

### Σύνδεσμοι (Links)
- **Δημιουργία υπερσυνδέσμου:**
  ```html
  <a href="https://www.example.com">Επισκεφτείτε μας</a>
  ```
- **Άνοιγμα συνδέσμου σε νέα καρτέλα:**
  ```html
  <a href="https://www.example.com" target="_blank">Νέο Παράθυρο</a>
  ```

### Εικόνες
- **Εισαγωγή εικόνας:**
  ```html
  <img src="εικόνα.jpg" alt="Περιγραφή εικόνας">
  ```
- **Μέγεθος εικόνας:**
  ```html
  <img src="εικόνα.jpg" width="300" height="200">
  ```

Το tag `<img>`, παίρνει τις εξής παραµέτρους:

- **src="images/tower.jpg"** : Η παράµετρος src παίρνει ως τιµή την απόλυτη διαδροµή και το όνοµα της εικόνας που θα εµφανιστεί. Σηµειώνεται πως οι εικόνες που υποστηρίζει ένας web browser είναι της µορφής jpg, gif και png.

- **width="400" height="267"**: Η παράµετρος width παίρνει ως τιµή το πλάτος της εικόνας σε pixels, ενώ η παράµετρος height το ύψος της εικόνας σε pixels. Σηµειώνεται πως δεν είναι απαραίτητο αλλά είναι καλό να χρησιµοποιούνται και επίσης πως αν αυξηθούν ή µειωθούν σε σχέση µε το πραγµατικό µέγεθος της εικόνας τότε η εικόνα θα µεγαλώσει ή µικρύνει αντίστοιχα. Υπάρχουν περιπτώσεις όπου για σχεδιαστικούς λόγους το πλάτος και ύψος µιας εικόνας πρέπει να δηλωθεί ως ποσοστό π.χ. 100%.

Αν για παράδειγµα θέλαµε να εισάγουµε την παρακάτω µία εικόνα που ονοµάζεται tower.jpg στην ιστοσελίδα µας ο κώδικας θα ήταν.

```html
<img src="images/tower.jpg" width="400" height="267">
```

µε δεδοµένο ότι η εικόνα βρίσκεται σε ένα φάκελο images και το αρχείο html βρίσκεται εκτός του φακέλου.

#### ∆ηµιουργία links σε µία εικόνα

Αν θέλουµε να δηµιουργήσουµε κάποιο Link σε µία εικόνα, ώστε όταν ένας χρήστης πατάει πάνω της να ανοίγει µία ιστοσελίδα, τότε χρησιµοποιούµε το tag `<a>` όπως περιγράψαµε παραπάνω για το κείµενο. Η µόνη διαφορά είναι ότι περιεχόµενο του tag `<a>` δεν θα είναι κείµενο αλλά ο κώδικας που εµφανίζει µία εικόνα.

Π.χ.
```html
<a href="https://thessaloniki.gr/">
<img src="images/tower.jpg" width="400" height="267" border="0">
</a>
```

`<figure>`: Χρησιμοποιείται για να ομαδοποιεί πολυμεσικά στοιχεία, όπως εικόνες ή διαγράμματα, μαζί με μια λεζάντα.

```html
<figure>
<img src="image.jpg" alt="Περιγραφή">
<figcaption>Λεζάντα εικόνας</figcaption>
</figure>
<figcaption>: Παρέχει λεζάντα για το στοιχείο <figure>.
```

`<svg>`: Χρησιμοποιείται για τη δημιουργία κλιμακωτών γραφικών vector.

```html
<svg width="100" height="100">
<circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />
</svg>
```

Name: Αντιστοιχεί ένα όνοµα στην φόρµα µας. Είναι χρήσιµο όταν αναφερόµαστε στη
φόρµα µέσα από διάφορα scripts (π.χ. javascript, jsp,..)

από διάφορα tags.
<label>: Συνδέεται με ένα στοιχείο εισαγωγής, προσφέροντας μια περιγραφή για το

<button>: Δημιουργεί ένα κουμπί το οποίο μπορεί να χρησιμοποιηθεί για την υποβολή μιας

  ```html

      <source src="video.mp4" type="video/mp4">
      Το πρόγραμμα περιήγησής σας δεν υποστηρίζει την προβολή βίντεο.
  </video>
  ```

  ```html
  <audio controls>
      <source src="audio.mp3" type="audio/mpeg">

  </audio>
  ```

Global Attributes
Τα Global Attributes είναι χαρακτηριστικά που μπορούν να χρησιμοποιηθούν σε όλα τα
HTML στοιχεία, ανεξαρτήτως του τύπου τους. Αυτά τα χαρακτηριστικά προσφέρουν
γενικές ιδιότητες που μπορούν να εφαρμόσουν σε οποιοδήποτε στοιχείο, προσφέροντας
ευελιξία και δυνατότητες προσαρμογής.
www.w3.org/TR/html/dom.html#global-attributes
Κύρια Global Attributes
id: Ορίζει ένα μοναδικό αναγνωριστικό για το στοιχείο. Χρησιμοποιείται κυρίως για να
γίνεται αναφορά στο στοιχείο μέσω CSS ή JavaScript.
<div id="header">This is the header</div>
class: Ορίζει μία ή περισσότερες κλάσεις για το στοιχείο, που επιτρέπουν την εφαρμογή
συγκεκριμένων στυλ μέσω CSS.
<p class="intro">This is an introduction paragraph.</p>
style: Εφαρμόζει inline στυλ στο στοιχείο χρησιμοποιώντας CSS.
<h1 style="color:blue;">This is a blue heading</h1>
lang: Καθορίζει τη γλώσσα του περιεχομένου του στοιχείου.
<p lang="en">Hello, World!</p>
title: Παρέχει πρόσθετες πληροφορίες για το στοιχείο, που εμφανίζονται ως tooltip όταν ο
χρήστης τοποθετεί τον δείκτη του πάνω από το στοιχείο.
<abbr title="HyperText Markup Language">HTML</abbr>
Event-handler Attributes
Τα Event-handler attributes επιτρέπουν την εκτέλεση JavaScript κώδικα όταν συμβαίνουν
συγκεκριμένα γεγονότα (events). Αυτά τα χαρακτηριστικά επιτρέπουν την αλληλεπίδραση
με τα στοιχεία της σελίδας με τη χρήση JavaScript.
onchange: Εκτελείται όταν η τιμή ενός στοιχείου αλλάζει.
<input type="text" onchange="alert('Changed!')">
onclick: Εκτελείται όταν ο χρήστης κάνει κλικ πάνω στο στοιχείο.
<button onclick="alert('Clicked!')">Click me</button>onmouseover: Εκτελείται όταν ο δείκτης του ποντικιού περνάει πάνω από το στοιχείο.
<div onmouseover="this.style.backgroundColor='yellow'">Hover over me!</div>
Mobile (Responsive Design)
Το Responsive Design αφορά τη δημιουργία ιστοσελίδων που προσαρμόζονται σε
διαφορετικά μεγέθη οθόνης και συσκευές, όπως κινητά τηλέφωνα, tablets, και
επιτραπέζιοι υπολογιστές. Ένας από τους βασικούς τρόπους για να επιτευχθεί αυτό είναι με
τη χρήση του meta tag viewport.
Meta Tag: Viewport
Το <meta name="viewport"> tag βοηθά στον έλεγχο του πώς η ιστοσελίδα σας
εμφανίζεται σε διαφορετικές συσκευές.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Κύριες Ιδιότητες του Viewport
●
●
●
●
●
width: Ορίζει το πλάτος της οθόνης. Μπορεί να είναι ένας ακέραιος αριθμός ή
device-width, που αντιστοιχεί στο πλάτος της συσκευής.
initial-scale: Ορίζει το αρχικό επίπεδο ζουμ της σελίδας. Τιμή 1.0 σημαίνει ότι η
σελίδα θα εμφανίζεται στο 100% του μεγέθους της.
minimum-scale: Ορίζει το ελάχιστο επιτρεπτό επίπεδο ζουμ της σελίδας. Αν είναι
1.0 αποτρέπεται το zoom-out.
maximum-scale: Ορίζει το μέγιστο επιτρεπτό επίπεδο ζουμ της σελίδας. Αν είναι 1.0
αποτρέπεται το zoom-in.
user-scalable: Ορίζει αν ο χρήστης μπορεί να κάνει ζουμ στη σελίδα. Αν η τιμή είναι
no, τότε το ζουμ αποτρέπεται.
Χρησιμοποιώντας αυτά τα χαρακτηριστικά, οι προγραμματιστές μπορούν να
δημιουργήσουν σελίδες που προσαρμόζονται σωστά σε κάθε συσκευή, προσφέροντας
έτσι μια καλύτερη εμπειρία χρήστη σε όλες τις πλατφόρμες​.
